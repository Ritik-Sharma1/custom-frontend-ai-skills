# Contributing to Custom Frontend AI Skills

Thank you for your interest in contributing to the Custom Frontend AI Skills repository! This project aims to expand the capabilities of agentic AI systems (specifically Claude and Model Context Protocol integrations) for UI/UX engineering, animations, and frontend best practices.

## How Can You Contribute?

We welcome contributions in the following areas:

1. **New AI Skills**: Submitting new `.md` instructional protocols for undocumented design systems or workflows.
2. **MCP Integrations**: Expanding our integration with FireCrawl, Google Stitch, or other context protocols.
3. **Refining Prompts**: Improving the system prompts within existing skills for better AI output accuracy.
4. **Bug Fixes**: Fixing syntax errors or improving documentation.

## Contribution Guidelines

1. **Fork the Repository:** Start by forking this repository to your GitHub account.
2. **Create a Branch:** Create a branch for your feature or bug fix (`git checkout -b feature/your-feature-name`).
3. **Commit Your Changes:** Write clear, concise commit messages.
4. **Submit a Pull Request:** Open a PR against the `main` branch. Provide a detailed description of what your PR solves or adds.

## Community

This project thrives on the idea that AI and frontend engineering are rapidly merging. By open-sourcing these agentic instructions, we hope to build a standard library of UI/UX capabilities for LLMs.
