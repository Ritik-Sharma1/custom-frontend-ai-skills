# Custom Frontend AI Skills

A curated collection of advanced agentic AI skills designed to turbocharge website development, UI/UX design, and brand extraction. Built with custom capabilities bridging modern frontend frameworks (React, Next.js, Tailwind), animation libraries (Anime.js), browser automation (Playwright), and web scraping (FireCrawl).

## Repository Contents

This repository includes a comprehensive set of AI agent skills, primarily tailored to enhance the quality and aesthetic of web applications and streamline developer workflows.

### Design & UI/UX
- **`ui-ux-pro-max`** - Extremely advanced UI/UX design intelligence handling design systems, data components, and framework specifics.
- **`frontend-design`** - Evaluates design feasibility, promotes strong aesthetic points-of-view, and creates non-generic visual identities.
- **`design-spells`** - Curated micro-interactions and high-craft design elements to add personality.
- **`stitch-ui-design`** - Specialized prompt crafting for integrating with Google Stitch AI UI design tools.
- **`tailwind-design-system`** & **`tailwind-patterns`** - Building scalable utility-class architectures, custom tokens, and enforcing v4 CSS-first best practices.

### Web Development & Generation
- **`build-a-website`** - The master orchestration skill to execute highly polished, production-ready sites from scratch.
- **`landing-page-generator`** - Optimizes for high-converting marketing materials with built-in PAS, AIDA, and BAB frameworks.
- **`final-website-refiner`** - Specifically trained to transform raw code into ultra-premium deployed sites utilizing FireCrawl for customized brand alignment.
- **`nextjs-react-expert`** & **`react-patterns`** - Enforces React best practices, eliminates waterfalls, and optimizes client/server boundaries.

### Animation & Performance
- **`animejs-animation`** - Creates complex, performant javascript orchestration for state-driven UI motion.
- **`fixing-motion-performance`** - Identifies and resolves layout thrashing and transition jank.

### Validation & QA
- **`ui-visual-validator`** - Conducts rigorous visual audits based on design system compliance and accessibility verification.
- **`website-qa`** - Validates web pages for broken links, responsive breakpoints, HTML structure, and cross-browser consistency.
- **`playwright-skill`** - Hands-on browser automation for local E2E UX confirmation, server detection, and form filling.

### Marketing & Brand Strategy
- **`brand-guidelines`** & **`copywriting`** - Maintains plain-speech vs. characteristic brand voice while converting passive visitors into leads.

## Usage

These skills are designed to be run as `.md` file extensions paired with supporting scripts that augment AI Assistants.

1. Ensure the required Agent/MCP runners are configured in your `.gemini/antigravity/skills/` dir.
2. Select the `SKILL.md` from the appropriate directory based on your current phase of development.
3. Execute the respective scripts via your CLI or AI prompt to streamline tasks and build production-grade interfaces.
